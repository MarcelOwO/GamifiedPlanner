## 3. Software Architektur

Die Applikation ist eine Android-Applikation gebaut mit der Sprache Kotlin. Die angewandte Architektur is MVVM. Wobei der Code in Layern aufgeteilt ist, naeherest dazu spaeter.

Verwendete Libraries sind in meinem Fall Koin fuer Dependency Injection und Firebase/Firestore fuer die Datenbank und Authentication.
